# Cipher Module Documentation

The Cipher module provides various encryption algorithms and encoding functions, including AES, 3DES, RSA encryption algorithms, MD5, SHA hash algorithms, and Base64 and Hex encoding functions.

## Component List

- [Encrypt (Encryption Class)](./encrypt.md) - Implementation of multiple encryption algorithms
- [Hasher (Hash Class)](./hasher.md) - Implementation of multiple hash algorithms
- [Encoder (Encoding Class)](./encoder.md) - Base64 and Hex encoding/decoding functions
